# Binary Tree Constructor

Build a binary tree visually and get the array you need to paste straight into LeetCode.

**Try it:** [https://binary-tree-constructor.vercel.app/](https://binary-tree-constructor.vercel.app/)

---

**Note:** This is a side project and totally vibe coded — no formal spec, just built iteratively. Use at your own risk; happy to accept improvements.

---

## What it does

- **Paste or type** a BFS array (e.g. `[1,2,3,null,4,5]`) and see the tree.
- **Draw the tree** by dragging nodes from the bottom input onto the canvas; drop left/right of a parent to set children.
- **Drag existing nodes** to reparent them (with their subtree).
- **Traversals:** Inorder, Preorder, Postorder with step-by-step highlight.
- **Validation:** BST check, balanced check, height.
- **Undo / Redo** (toolbar + Ctrl+Z / Ctrl+Shift+Z).
- **Delete** selected node (button or Delete/Backspace key); Escape clears selection.
- **Copy** the array or export as JSON.
- **Sample trees** dropdown to load preset trees.
- **Themes:** Dark, Light, LeetCode (default).
- **Zoom** inside the canvas (Ctrl/Cmd + wheel or pinch); **pan** by dragging the background. Canvas is resizable.

That’s it — no sign-up, just build your tree and copy the input.
