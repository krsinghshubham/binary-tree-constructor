# Binary Tree Constructor

Build a binary tree visually and get the array you need to paste straight into LeetCode.

**Try it:** [https://binary-tree-constructor.vercel.app/](https://binary-tree-constructor.vercel.app/)

- Draw the tree by dragging nodes, or paste an existing array to see it as a tree.
- Edit nodes, delete them, or move subtrees by dragging.
- Copy the result and paste it into LeetCode as the tree input.

That’s it — no technical setup, just build your tree and copy the input.
